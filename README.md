## About The Project

[![Product Name Screen Shot][product-screenshot]](https://thepandadevs.github.io/taco-nochero/)

Application with design UX/UI oriented a Mexican restaurant to our subject User experience, where you will find frames like:

- Landing
- Products list
- Detail
- User profile
- Login
- Register
- Pricing
- Error pages
- Responsive design
- Shopping cart
- Payment view
- Checkout
- Chat
- Notifications
- Inbox email
- Forms
    - Form elements
    - Wizzard
    - Pickers
    - Validations
    - Datatables
- Dashboard
    - Charts
    - Products list
    - Register

## Getting Started

This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

-   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/ADKAN-UTEZ/taco-nochero.git
    ```
3. Install NPM packages
    ```sh
    npm install
    ```
4. Enter command
    ```sh
    npm start
    ```
